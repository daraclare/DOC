WebType

This webpage illustrates the use of the jQuery plugin 'FitText' which makes font-sizes flexible. It should be used on fluid or responsive layouts to achieve scalable headlines that fill the width of a parent element.

This webpage also illustrates the use of the jQuery plugin 'Lettering'.

TypeKit is used to implement the fonts 'Lobster' or 'Droid Sans' for headings and 'Droid Serif' for body.
